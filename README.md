# Alc::Cmd

Command-line tool to translate English to Japanese 

## Installation

Add this line to your application's Gemfile:

    gem 'alc-cmd'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install alc-cmd

## Usage

    $ alc-cmd s search word
